What is JS
JS is one of the core technologies of the web, alongside with HTML and CSS

Is a high leve interpreted programming language, can be user in the client side (frontend) and in server side (backend) with NodeJS.

What JS is used for:
- DOM manipulation.
- Event handling.
- Asynchronous requests.
- Animations and effects.
- Data manipulation (sorting, filtering)
- Storing data (cookies and localStorage)
- Single page applications (SPA)
- Creating APIS & WS (NodeJS and Deno)
